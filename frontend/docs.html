Rock API - Frontend

Bem-vindo à documentação do frontend da Rock API! Este sistema foi desenvolvido para facilitar a gestão de bandas, shows e participações em eventos musicais.

Sobre o Projeto: Esta API REST didática foi desenvolvida para ensinar, de forma simples e prática, os principais conceitos da arquitetura REST e do padrão MVC.

Características Principais

Visualização de Dados
  • Tabelas responsivas
  • Ordenação por colunas
  • Paginação inteligente
  • Busca em tempo real
Manipulação de Dados
  • Criação de registros
  • Edição inline
  • Exclusão segura
  • Validação de formulários
Autenticação
  • Login seguro
  • Registro de usuários
  • Controle de acesso
  • Diferentes perfis de usuário
Exportação
  • Exportação em CSV
  • Exportação em JSON
  • Exportação em XML
  • Download direto

Como Usar

  1. Acesse a página de login
  2. Se não tiver conta, clique em "Criar Conta"
  3. Preencha os dados e selecione seu tipo de usuário:
    • Músico: Pode gerenciar bandas e participações
    • Organizador: Pode gerenciar shows e eventos
    • Admin: Acesso completo ao sistema
  4. Faça login com suas credenciais

Use o menu superior para navegar entre as seções:

  • Dashboard: Visão geral com estatísticas e ações rápidas
  • Bandas: Gestão completa de bandas de rock
  • Shows: Gerenciamento de eventos e apresentações
  • Participações: Vinculação de bandas a shows

  • Busca: Digite na barra de busca para filtrar resultados
  • Ordenação: Clique nos cabeçalhos das tabelas para ordenar
  • Paginação: Use os botões na parte inferior das tabelas
  • Ações: Use os botões "Editar" e "Excluir" em cada linha
  • Modais: Formulários aparecem em janelas modais para melhor UX

Gestão de Bandas

A seção de bandas permite gerenciar todas as informações sobre as bandas de rock.

Campos Disponíveis:

Campo Tipo Obrigatório Descrição
Nome Texto Nome da banda
País de Origem Texto País onde a banda foi formada
Ano de Formação Número Ano em que a banda foi formada
Gênero Musical Texto Estilo musical da banda

Gestão de Shows

Gerencie eventos, concertos e apresentações musicais.

Campos Disponíveis:

Campo Tipo Obrigatório Descrição
Local Texto Local onde o show será realizado
Data Data Data de realização do show
Público Estimado Número Quantidade estimada de público

Gestão de Participações

Vincule bandas a shows, definindo ordem de apresentação e tempo de execução.

Campos Disponíveis:

Campo Tipo Obrigatório Descrição
Banda Seleção Banda que participará do show
Show Seleção Show em que a banda participará
Ordem de Apresentação Número Ordem em que a banda se apresentará
Tempo de Execução Número Duração da apresentação em minutos

Sistema de Autenticação

Importante: É necessário estar logado para usar as funcionalidades do sistema.

Tipos de Usuário:

Músico

Pode gerenciar bandas e criar participações.

Organizador

Pode gerenciar shows e eventos musicais.

Admin

Acesso completo a todas as funcionalidades.

Exportação de Dados

O sistema permite exportar dados em diferentes formatos:

CSV

Ideal para planilhas (Excel, Google Sheets)

JSON

Para integração com outros sistemas

XML

Formato estruturado para intercâmbio

Solução de Problemas

  • Verifique se o email e senha estão corretos
  • Certifique-se de que possui uma conta registrada
  • Limpe o cache do navegador
  • Verifique a conexão com a internet

  • Atualize a página (F5 ou Ctrl+R)
  • Verifique sua conexão com a internet
  • Verifique se o token de autenticação não expirou
  • Faça logout e login novamente

  • Verifique se todos os campos obrigatórios foram preenchidos
  • Certifique-se de que possui permissão para a ação
  • Verifique se os dados inseridos são válidos
  • Tente novamente após alguns segundos
