1. Introdução

O Rock Manager é uma interface web completa e moderna para gerenciar bandas, shows e participações em eventos musicais. O sistema foi desenvolvido utilizando tecnologias web padrão e o framework Bootstrap para garantir uma experiência responsiva e intuitiva.

Principais Funcionalidades:
  • Dashboard Interativo: Visão geral das estatísticas e atividades recentes
  • Gerenciamento de Bandas: CRUD completo com busca e filtros avançados
  • Gerenciamento de Shows: Criação e controle de eventos musicais
  • Participações: Vinculação de bandas aos shows com controle de ordem e duração
  • Autenticação: Sistema de login com diferentes níveis de permissão
  • Import/Export: Funcionalidades para importar e exportar dados em CSV, JSON e XML
  • Design Responsivo: Interface adaptável para desktop, tablet e smartphone

2. Primeiros Passos

3. Autenticação

Sistema de Login:

O sistema utiliza autenticação baseada em JWT (JSON Web Tokens) para garantir a segurança dos dados.

Tipos de Usuário:
Tipo Permissões Descrição
Admin Acesso total Pode criar, editar e excluir qualquer registro
Músico Bandas e participações Pode gerenciar bandas e suas participações em shows
Organizador Shows e participações Pode criar shows e gerenciar participações
Usuário Apenas visualização Pode visualizar dados mas não pode modificar
Registro de Novo Usuário:
  1. Na página de login, clique em "Criar conta"
  2. Preencha os dados solicitados
  3. Escolha o tipo de usuário apropriado
  4. Após o registro, faça login com as credenciais criadas

4. Dashboard

O Dashboard é a página principal do sistema, oferecendo uma visão geral dos dados e ações rápidas.

Elementos do Dashboard:
  • Cartões de Estatísticas: Mostram contadores de bandas, shows, participações e usuários
  • Ações Rápidas: Botões para acessar rapidamente as principais funcionalidades
  • Atividade Recente: Lista das últimas bandas e shows cadastrados
  • Navegação: Menu principal para acessar todas as seções do sistema
Dica: O Dashboard é atualizado automaticamente a cada 5 minutos, mas você pode clicar no botão "Atualizar" nas páginas específicas para obter dados mais recentes.

5. Gerenciamento de Bandas

Funcionalidades Disponíveis:
Criar Nova Banda:
  1. Clique no botão "Nova Banda"
  2. Preencha as informações obrigatórias (nome é obrigatório)
  3. Adicione informações opcionais como gênero, país e ano de formação
  4. Clique em "Salvar" para cadastrar a banda
Editar Banda:
  1. Na lista de bandas, clique no ícone de edição (lápis)
  2. Modifique as informações desejadas
  3. Clique em "Atualizar" para salvar as alterações
Excluir Banda:
  1. Clique no ícone de exclusão (lixeira) na linha da banda
  2. Confirme a exclusão no modal que aparece
  3. Atenção: A exclusão é irreversível
Importante: Para criar, editar ou excluir bandas, você precisa ter permissões de Músico ou Admin.

6. Gerenciamento de Shows

Criação de Shows:
  1. Clique em "Novo Show"
  2. Preencha o local (obrigatório) e a data (obrigatória)
  3. Opcionalmente, adicione o público estimado
  4. Salve o show para criá-lo
Status dos Shows:
  • Agendado - Shows futuros
  • Hoje - Shows que acontecem hoje
  • Realizado - Shows que já aconteceram
Visualização de Detalhes:

Clique no ícone de visualização (olho) para ver informações detalhadas do show, incluindo a opção de visualizar as participações relacionadas.

Dica: Shows passados não podem ser editados, apenas visualizados para manter a integridade histórica dos dados.

7. Gerenciamento de Participações

O que são Participações:

Participações representam a relação entre bandas e shows, definindo qual banda tocará em qual show, sua ordem de apresentação e duração.

Criação de Participação:
  1. Clique em "Nova Participação"
  2. Selecione a banda e o show
  3. Defina a ordem de apresentação (posição da banda no show)
  4. Especifique a duração em minutos
  5. Salve a participação
Informações Exibidas:
  • Banda: Nome e gênero musical
  • Show: Local e público estimado
  • Data: Data do show com indicador se é hoje
  • Ordem: Posição da banda na apresentação
  • Duração: Tempo de apresentação em minutos
  • Status: Agendado, hoje ou realizado
Regra: Uma banda não pode participar do mesmo show duas vezes.

8. Busca e Filtros

Funcionalidades de Busca:
Busca por Texto:
  • Bandas: Busca por nome, gênero ou país
  • Shows: Busca por local
  • Participações: Busca por nome da banda ou local do show
Filtros Avançados:
  • Bandas: Filtrar por gênero e país de origem
  • Shows: Filtrar por local específico e mês/ano
  • Participações: Filtrar por banda específica ou show específico
Ordenação:

Clique nos cabeçalhos das colunas para ordenar os dados. Uma segunda clique inverte a ordem (crescente/decrescente).

Paginação:

Use os controles de paginação na parte inferior das tabelas. Você pode alterar o número de itens por página (10, 25, 50 ou 100).

Desempenho: A busca é otimizada e funciona em tempo real conforme você digita.

9. Importação e Exportação

Formatos Suportados:
  • CSV: Formato de planilha, compatível com Excel
  • JSON: Formato estruturado para desenvolvedores
  • XML: Formato estruturado padrão
Exportação de Dados:
  1. Na página desejada (Bandas, Shows ou Participações)
  2. Clique no botão "Exportar"
  3. Escolha o formato desejado
  4. O arquivo será baixado automaticamente
Importação de Dados:
  1. Prepare seu arquivo no formato correto
  2. Use a seção "Importar" na página correspondente
  3. Selecione o arquivo
  4. Escolha o formato
  5. Clique em "Importar"
Estrutura CSV para Bandas:
id,nome,genero,pais_origem,ano_formacao
Estrutura CSV para Shows:
id,local,data,publico_estimado
Estrutura CSV para Participações:
id_banda,id_show,ordem_apresentacao,tempo_execucao_min

11. Uso em Dispositivos Móveis

Design Responsivo:

O sistema foi desenvolvido com design responsivo, adaptando-se automaticamente a diferentes tamanhos de tela.

Funcionalidades Mobile:
  • Menu Colapsível: Menu principal se transforma em hamburger menu
  • Tabelas Responsivas: Scroll horizontal automático em tabelas grandes
  • Botões Adaptados: Botões maiores e mais fáceis de tocar
  • Formulários Otimizados: Campos apropriados para toque
Dicas para Uso Mobile:
  • Use o modo paisagem para melhor visualização de tabelas
  • Os filtros podem ser acessados rolando horizontalmente
  • Modais se adaptam ao tamanho da tela automaticamente

12. Solução de Problemas

Problemas Comuns:

  • Verifique se o email e senha estão corretos
  • Tente usar uma das credenciais de demonstração
  • Verifique se há mensagens de erro específicas
  • Limpe o cache do navegador

  • Verifique sua conexão com a internet
  • Clique no botão "Atualizar" da página
  • Recarregue a página do navegador (F5)
  • Verifique se você está logado corretamente

  • Verifique se você tem as permissões necessárias
  • Confira se todos os campos obrigatórios estão preenchidos
  • Verifique se os dados inseridos são válidos
  • Tente fazer logout e login novamente

  • Verifique se o arquivo está no formato correto
  • Confirme se as colunas estão na ordem certa
  • Certifique-se de que não há caracteres especiais inválidos
  • Verifique se o arquivo não excede o tamanho máximo
Suporte: Se os problemas persistirem, entre em contato com o administrador do sistema ou consulte os logs de erro no console do navegador (F12).
